ProgramExecutionCounter
=======================

This tool will display information from UserAssist registry key where Windows 7 stores information about how many times a program has been executed.

Features
--------

*  Filter entries by type (program or shortcut) and/or name.

![Display all](http://thesolidsnake.files.wordpress.com/2014/02/gambar17.png)

![Search by name](http://thesolidsnake.files.wordpress.com/2014/02/gambar52.png)

*  Update execution count to registry.

![Update execution count](http://thesolidsnake.files.wordpress.com/2014/02/gambar27.png)

*  Delete entry from registry.

![Delete from registry](http://thesolidsnake.files.wordpress.com/2014/02/gambar34.png)

*  Detailed information.

![Detail popup](http://thesolidsnake.files.wordpress.com/2014/02/gambar43.png)
